# Tic Tac Toe (C)

A simple two-player Tic Tac Toe game written in C for the terminal.

## How to Run

```bash
gcc tictactoe.c -o tictactoe
./tictactoe
````

## Features

* 2-player mode (X and O)
* Input validation
* Win and draw detection

## Example Board

```
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```
